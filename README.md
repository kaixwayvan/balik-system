# BALIK: AI-Assisted Gamified Lost-and-Found System

## Project Overview
BALIK is a web-based lost-and-found management system designed for the
Polytechnic University of the Philippines (PUP) – Sta. Mesa Campus.
The system improves the traditional manual process by providing a
centralized, secure, and automated platform for reporting, matching,
and claiming lost items.

The system integrates:
- Natural Language Processing (NLP) for smart item matching
- QR Code verification for secure claiming
- Gamification features to encourage user participation
